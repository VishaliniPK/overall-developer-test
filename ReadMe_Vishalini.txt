Steps to follow prior to running the applications :
1. Restore the backup database MauPhoto.bak or create a new daatabase, name : MauPhoto, and then run the provided script.
2. Deploy the published version of both the WebAPI and WebSite on IIS.
3. For the WebSite, modify the Web.config for the key MauPhotoWebAPI and upload the value with the link of the published version of the WebAPI.
